## 📝 Common Tasks

### Register a Survivor
1. Click **"Survivors"** → **"New Intake"**
2. Fill in personal info
3. Add medical needs
4. Review and submit
5. ✅ Gets case number automatically

### Create Disaster Event
1. Click **"Disasters"** → **"+ New Disaster"**
2. Enter name, type, severity
3. Add affected area
4. Click **"Create Event"**

### Add Volunteer
1. Click **"Volunteers"** → **"+ Add Volunteer"**
2. Enter name, contact, skills
3. Select skills from list
4. Click **"Add Volunteer"**

### Record Distribution
1. Click **"Distributions"** → **"+ Record Distribution"**
2. Select disaster, resource, quantity
3. Choose distribution type
4. Click **"Record Distribution"**

### Create Emergency Request
1. Click **"Emergency Dispatch"** → **"+ New Request"**
2. Select type, priority, location
3. Add description
4. Click **"Submit Request"**
5. Assign volunteers from right panel

## 🔐 Logout (Fixed!)

**To logout properly:**

1. Look at **bottom-left** of sidebar
2. Find your **profile section**
3. Click the **🚪 Logout icon**
4. You'll go to login page

⚠️ **If still seeing dashboard:**
- Press `Ctrl + Shift + Delete`
- Clear "Cookies and site data"
- Click "Clear data"
- Try logout again
