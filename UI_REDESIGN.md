## 🎨 Design System

### Color Palette

| Purpose | Color | Usage |
|---------|-------|-------|
| Emergency | Red (#DC2626) | Help requests, urgent actions |
| Information | Blue (#2563EB) | Registration, info sections |
| Success | Green (#16A34A) | Confirmation, tracking |
| Progress | Purple (#9333EA) | In-progress states |
| Warning | Yellow (#EAB308) | Pending states |

### Typography

| Element | Size | Weight |
|---------|------|--------|
| Headings | 2xl-6xl | Bold |
| Body | base-lg | Normal |
| Labels | sm | Medium |
| Small Text | xs-sm | Normal |

### Components

**Cards:**
- White background
- No border
- Large shadow (shadow-2xl)
- Rounded corners (rounded-xl)

**Buttons:**
- Gradient backgrounds
- Large padding (py-6)
- Bold text
- Hover effects

**Icons:**
- Large (w-6 h-6, w-8 h-8)
- In gradient circles
- Consistent spacing

## ✨ Animations & Interactions

**Hover Effects:**
- Cards lift up (`-translate-y-1`)
- Shadows increase (`shadow-2xl`)
- Icons scale up (`scale-110`)
- Borders change color

**Transitions:**
- Smooth color changes
- Transform animations
- All 200-300ms duration

**Loading States:**
- Disabled buttons
- Loading text
- Spinner indicators
